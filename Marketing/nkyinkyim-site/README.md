# Nkyimkyim Woodworks Website

## Overview
This repository contains the source code for the **Nkyimkyim Woodworks** website — a one-page, responsive website designed to showcase the brand, services, and portfolio of a premium woodworking and interior installation business.

The website serves as:
- A digital presence for the business  
- A portfolio showcase of completed works  
- A contact and enquiry channel for potential clients  
- A supporting platform for offline marketing (billboards, flyers, social media)

---

## Business Description
**Nkyimkyim Woodworks** specializes in:
- Custom woodworking
- Interior installations
- Bespoke furniture
- Design, consultation, and build execution

The brand name *“Nkyimkyim”* represents **transformation**, which is reflected in the company’s approach to turning raw ideas and materials into refined interior spaces.

**Target audience:**
- Homeowners  
- Airbnb / short-let property owners  
- Commercial and retail clients  

**Service coverage:** Nationwide

---

## Website Features
- One-page scrolling layout
- Sticky navigation bar
- Hero section with clear value proposition
- Services overview
- Portfolio / gallery section
- Client testimonials
- Contact form
- Social media integration (WhatsApp, Facebook, Instagram, TikTok, X, Snapchat)
- Responsive design for desktop, tablet, and mobile devices

---

## Tech Stack
- **HTML5**
- **CSS3**
- No external frameworks or libraries
- No JavaScript (lightweight, fast-loading design)

This approach ensures:
- Fast load times  
- Simplicity and maintainability  
- Compatibility with most browsers
